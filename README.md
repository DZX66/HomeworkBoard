## 项目简介

HomeworkBoard 是一款基于 Electron 开发的桌面应用，专为学校一体机设计，用于高效管理各科作业任务。支持多科目分类管理、智能输入提示、自动保存和历史追溯等功能。

## 主要功能

- 📚 **多科目管理**：预置11个常见学科，支持自定义科目名称
- 📝 **智能输入**：
  - 中文括号快捷输入（交/不交）
  - 智能应用上一次输入内容
  - 回车创建新任务项
- ⏳ **历史记录**：
  - 每日自动记录
  - 历史记录日历视图
  - 支持查看任意日期作业详情
- 🔧 **实用工具**：
  - 界面缩放（50%-200%）
  - 剪贴板/文件导出
  - 自动保存（默认间隔10分钟）
- 🖥 **桌面集成**：
  - 系统级菜单
  - 配置文件


## 使用说明

### 基本操作
1. **新增任务**：在任意文本框按回车键
2. **删除任务**：清空内容后按退格键
3. **快速输入**：输入中文左括号"（"触发快捷选项
4. **修改标题**：点击主标题进入编辑模式

### 导出功能
- **剪贴板导出**：通过菜单栏 `文件 -> 导出 -> 复制到剪贴板`
- **文件导出**：通过菜单栏 `文件 -> 导出 -> 保存为文件`

### 历史记录
通过菜单栏 `查看 -> 历史记录` 打开日历视图：
- 点击日期查看当天作业详情
- 绿色日期表示有历史记录

## 配置选项

配置文件位于用户数据目录（可通过菜单栏打开）：
- `config.json`：应用配置
  - `subjects`：科目列表
  - `saveTime`：每日自动保存时间（分钟数，例如1220表示20:20）
  - `autoSaveGap`：自动保存间隔（分钟）


## 技术栈
- **前端**：Electron + HTML5/CSS3
- **数据存储**：SQLite3 + electron-store
- **构建工具**：electron-builder
